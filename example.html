This is an example to upload the file to git hub.....

Steps to be followed.....

  • Check the status of the folder does any changes are done...
  • If yes the it shows the changes in red color....
  • Now follow the 3 commands...
  • git add filename.extension
  • git commit -m comment on changes that are added...
  • git push
  • Now you can see this file on the github account...
